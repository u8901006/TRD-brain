# TRD Brain

**Treatment-Resistant Depression (TRD) Daily Literature Report**

AI-powered PubMed surveillance system that automatically fetches, analyzes, and publishes daily TRD literature reports.

## Features

- Automated daily PubMed search across 45+ TRD-relevant journals
- AI analysis with Zhipu GLM-5-Turbo (with fallback to GLM-4.7 / GLM-4.7-Flash)
- PICO analysis, clinical utility assessment, and topic classification
- Beautiful dark-themed HTML reports deployed to GitHub Pages
- Traditional Chinese (Taiwan) summaries for healthcare professionals

## Schedule

- Daily at 01:00 AM Taipei Time (UTC 17:00)
- Manual trigger available via GitHub Actions workflow_dispatch

## Topics Covered

- Treatment-resistant depression (TRD) core research
- Ketamine / Esketamine / Psychedelics
- rTMS / ECT / DBS / VNS / Neuromodulation
- Pharmacological augmentation strategies
- Biomarkers and neuroimaging
- Psychotherapy for difficult-to-treat depression
- Suicide risk and comorbidities
